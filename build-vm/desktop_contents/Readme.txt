Introduction:

TouchDevelop is a web-application to develop mobile applications. 
You can easily develop diverse types of mobile applications by using its simple syntaxt. 
The web-application functions as an IDE to develop apps through Touch based Interface.

Get Started With TouchDevelop:

In this VM, you should have TouchDevelop tool running locally on URL
"http://localhost:4242/editor/local/"

1) If browser is not open, open the above URL in browser

2) If URL is not functioning, run the executable script linked on desktop.

3) Once you can see the application, it is easy to get started by choosing from the list of tutorials available on the landing page.
Choose any of them and follow easy steps to create your mobile application.
